# Sculpt Fitness

A responsive landing page for a fitness & gym business — built with HTML, CSS, and vanilla JavaScript.

## Live Demo

https://chandan12ar.github.io/Sculpt-fitness/

## Features

- Hero image slider (Swiper.js)
- About, features, and pricing sections
- Trainer profiles with social links
- Client testimonials
- Blog/news section
- Fully responsive layout
- Font Awesome icons

## Tech Stack

- HTML5, CSS3, JavaScript
- Swiper.js v7 (slider/carousel)
- Font Awesome 5.15.4

## Getting Started

No build step required — clone the repo and open `index.html` in a browser.

```
git clone https://github.com/chandan12ar/Sculpt-fitness.git
cd Sculpt-fitness
open index.html
```
